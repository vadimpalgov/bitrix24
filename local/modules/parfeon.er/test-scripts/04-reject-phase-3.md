## Шаги

### Шаги 1–4 — аналогично сценарию 1

Пройти шаги 1–4 сценария 01: создание заявки → В работе → Сидорова одобряет → Орлов одобряет.  
После шага 4 у Дмитриева (ID 4) должен быть AP фазы 3, `UF_CRM_11_ORDER = 1`.

---

### Шаг 5а — Попытка отклонить без причины

**Действие:** войти под `dmitriev` (ID 4), перевести AP в `AP_REJECT_STATUS` **без заполнения** причины.

**Ожидаемый результат:**
- [ ] Сохранение не выполнено — ошибка «Не указана причина отклонения»
- [ ] AP Дмитриева остаётся в начальной стадии
- [ ] ER остаётся в `ER_START_STATUS`
- [ ] AP для Генералова **не создан**

---

### Шаг 5б — Отклонить с причиной

**Действие:** войти под `dmitriev` (ID 4), заполнить причину, перевести AP в `AP_REJECT_STATUS`.

**Ожидаемый результат:**
- [ ] AP Дмитриева → `AP_REJECT_STATUS`
- [ ] ER переходит в `ER_REJECT_STATUS`
- [ ] Иванов получил IM-уведомление об отклонении с причиной
- [ ] AP для Генералова (`director`, ID 34) **не создан** — цепочка прервана
